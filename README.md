# EyeSOC Threat Detection Dashboard

A lightweight SOC dashboard for small businesses to detect security threats from log files.

## Features
- Brute force detection
- Port scan detection
- Malware detection
- Suspicious IP detection
- Privileged account monitoring
- File upload and drag and drop support
- Risk level filtering

## How to run

Make sure you have Node.js installed, then:

git clone https://github.com/sherz111/eyesoc-dashboard.git
cd eyesoc-dashboard
npm install
npm run dev

Then open http://localhost:5173 in your browser.
